## eva-mat-in-boat.webp — фото для главной

Фотография **готового EVA-коврика, установленного в лодке**.

- формат: `.webp` (или замените расширение в `index.html`, если `.jpg`);
- ориентация: горизонтальная, соотношение ~4:3;
- рекомендуемый размер: ~1200×900, вес до ~200–300 КБ;
- имя файла строго: `eva-mat-in-boat.webp`.

Пока файла нет, на главной показывается аккуратный плейсхолдер (без «битой» картинки). Как только файл появится в этой папке и будет запушен — фото отобразится автоматически.
